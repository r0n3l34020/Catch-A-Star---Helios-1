# Helios-1: Solar Threat Analysis System

## 🌟 Project Overview
Helios-1 is an automated solar threat detection and analysis system designed to monitor active solar regions and assess potential risks to satellite infrastructure. This project utilizes computer vision to process solar telemetry, quantifying sunspot activity to provide actionable data for satellite safety.

## 🏆 Recognition
*   **Award:** Winner of the **EAAE *Catch a Star* 2025** competition.

## 🛠️ Technical Architecture
Helios-1 is built on a modular Python architecture designed for efficient data processing and visualization:

*   **Data Acquisition:** Automated utilities to fetch and process real-time solar imagery.
*   **Computer Vision Engine:** Advanced algorithms for precise sunspot measurement and region characterization.
*   **Threat Assessment:** Heuristic-based analysis to evaluate the risk level of detected active regions.
*   **Dashboard & Visualization:** A centralized UI to visualize solar telemetry and analytical outcomes.

## 🚀 Key Features
*   **Autonomous Monitoring:** Systematic data ingestion from solar archives.
*   **Quantifiable Analytics:** Precise sunspot dimension and threat-level metrics.
*   **Unified Dashboard:** Integrated UI for real-time risk visualization.
